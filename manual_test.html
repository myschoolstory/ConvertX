Manual Testing Instructions for Image Converter Application

This document provides step-by-step instructions for manually testing the Image Converter application.

Prerequisites

  • The application is running at http://localhost:5177/
  • Test image file is available at /app/test.jpg

Test Cases

Test Case 1: Upload Functionality for JPG/JPEG Files

  1. Open the application at http://localhost:5177/
  2. Click on the "Choose File" button or the upload area
  3. Select the test image file /app/test.jpg
  4. Verify that the image is displayed in the preview area

Expected Result: The image should be successfully uploaded and displayed in the preview area.

Result: PASS / FAIL

Notes:

Test Case 2: Drag and Drop Functionality for JPG/JPEG Files

  1. Open the application at http://localhost:5177/
  2. If there's already an image loaded, click the "Clear" button
  3. Drag the test image file /app/test.jpg from your file explorer and drop it onto the drop zone
  4. Verify that the image is displayed in the preview area

Expected Result: The image should be successfully uploaded via drag and drop and displayed in the preview area.

Result: PASS / FAIL

Notes:

Test Case 3: Conversion Settings (Compression Quality and Lossless Mode)

  1. Upload the test image using either method from Test Case 1 or 2
  2. Locate the quality slider and adjust it to different values (e.g., 50%, 75%, 100%)
  3. Toggle the "Lossless" checkbox on and off
  4. Click the "Convert" button after each change
  5. Verify that the conversion settings affect the output image quality and file size

Expected Result: The conversion settings should affect the output image quality and file size. Higher quality settings should result in larger file sizes but better image quality.

Result: PASS / FAIL

Notes:

Test Case 4: Image Preview Before and After Conversion

  1. Upload the test image using either method from Test Case 1 or 2
  2. Verify that the original image is displayed in the preview area
  3. Click the "Convert" button
  4. Verify that both the original and converted images are displayed side by side or in a before/after view
  5. Check if there's a comparison feature that allows toggling between the original and converted images

Expected Result: The application should display both the original and converted images, allowing the user to compare them.

Result: PASS / FAIL

Notes:

Test Case 5: Download of the Converted PNG Image

  1. Upload the test image using either method from Test Case 1 or 2
  2. Click the "Convert" button
  3. Once the conversion is complete, click the "Download PNG" button
  4. Verify that the file is downloaded successfully
  5. Open the downloaded file to confirm it's a valid PNG image

Expected Result: The converted PNG image should be downloaded successfully and should be a valid PNG file.

Result: PASS / FAIL
